# Plan: Language Selection + Auto-fill AI Results

## 1. Add `language` field to social accounts
- Default: `fa` (Persian)
- Options: fa, en, ar, tr, etc.
- Used in AI prompt to generate content in that language

## 2. Auto-fill from AI results
After AI analysis completes:
- `user_caption` ← AI generated caption (auto-replace)
- `text_on_video` (static_text) ← AI generated title
- `watermark` ← social account username (e.g. @dknow)

## Implementation

### Migration
```sql
ALTER TABLE zernio_social_accounts ADD COLUMN language TEXT DEFAULT 'fa';
```

### AI Prompt (index.ts)
```ts
const language = socialAccount?.language || 'fa';
const langNames = { fa: 'Persian/Farsi', en: 'English', ar: 'Arabic', tr: 'Turkish' };
// Add to system prompt:
// Generate all content in ${langNames[language]} language.
```

### Auto-fill logic (index.ts callback)
After saving AI results, also update video fields:
```ts
await InstagramVideo.update(video.id, {
    user_caption: aiResult.caption,
    text_on_video: aiResult.title,
    watermark: socialAccount?.username ? `@${socialAccount.username}` : null,
});
```

### Frontend (VideoEdit.tsx)
After AI results arrive via poll, auto-set form values:
```ts
form.setFieldValue('user_caption', data.ai_caption);
form.setFieldValue('static_text', data.ai_title);
form.setFieldValue('watermark_text', `@${username}`);
```
