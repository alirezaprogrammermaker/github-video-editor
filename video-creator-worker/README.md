# پنل مدیریت ربات تلگرام

پنل مدیریت جامع برای ربات‌های تلگرام با قابلیت هوش مصنوعی، ساخته شده روی Cloudflare Workers. این پروژه یک اپلیکیشن وب کامل (SPA + API) است که به شما امکان مدیریت کاربران، کانال‌ها، سیستم راهنما و چت‌بات هوش مصنوعی ربات تلگرام را می‌دهد.

## ویژگی‌ها

### احراز هویت
- ثبت‌نام و ورود با ایمیل و رمز عبور
- هش رمز عبور با PBKDF2 (100,000 iterations, SHA-256)
- نشست ۷ روزه با کوکی امن (HttpOnly, Secure, SameSite=Strict)
- قابلیت فعال/غیرفعال‌سازی ثبت‌نام از تنظیمات

### داشبورد آماری
- نمایش تعداد کل کاربران، امروز، دیروز، این هفته و این ماه
- نمودار خطی روند ثبت‌نام کاربران (۷، ۱۴، ۳۰، ۹۰ روز)
- نمودار ستونی ثبت‌نام روزانه (۱۴ روز اخیر)

### مدیریت کاربران تلگرام
- جدول کامل مدیریت کاربران (chat_id، نام کاربری، نام، نقش، وضعیت، تاریخ عضویت)
- جستجو و فیلتر کاربران
- تغییر نقش کاربر (کاربر / مدیر)
- مسدود کردن کاربران با دلیل و مدت زمان (مسدودی خودکار منقضی می‌شود)
- ارسال پیام مستقیم به هر کاربر (متن ساده، HTML، Markdown)
- حذف کاربران

### مدیریت نشست‌های کاربران
- مشاهده تمام نشست‌های مکالمه با کاربران تلگرام
- فیلتر بر اساس وضعیت: فعال، تکمیل شده، لغو شده
- مشاهده جزئیات نشست با داده‌های JSON
- لغو و حذف نشست‌ها

### مدیریت کانال‌های تلگرام
- اضافه کردن کانال‌ها و گروه‌ها با نام کاربری
- اعتبارسنجی وجود کانال از طریق API تلگرام
- فعال/غیرفعال‌سازی اشتراک اجباری در کانال
- بررسی وضعیت ادمین ربات در کانال اجباری

### سیستم راهنمای ربات
- CRUD آیتم‌های راهنما (نام، توضیحات با پشتیبانی HTML، ترتیب نمایش)
- نمایش دکمه‌های اینلاین کیبورد به کاربران تلگرام
- تزریق خودکار قوانین به پرامپت هوش مصنوعی

### هوش مصنوعی (Cloudflare Workers AI)
- دو پیکربندی هوش مصنوعی مجزا: **مدیر** و **کاربر**
- مدل‌های پشتیبانی شده:
  - Llama 4 Scout 17B (پیش‌فرض)
  - Llama 3.3 70B (بالاترین کیفیت)
  - Mistral Small 3.1 24B
  - Llama 3.2 3B (سبک و سریع)
  - Qwen3 30B
  - DeepSeek R1 32B
- تنظیم پرامپت سیستم، توکن‌های حداکثر، دما و محدودیت روزانه
- ثبت و پیگیری مصرف هوش مصنوعی (توکن و درخواست)
- رابط چت آزمایشی هوش مصنوعی در داشبورد
- دسترسی کنترل‌شده به پایگاه داده توسط هوش مصنوعی (فهرست مجاز جداول)

### مدیریت ربات
- تنظیم توکن ربات با اعتبارسنجی از API تلگرام
- تنظیم، حذف و مشاهده اطلاعات Webhook
- تنظیم نام ربات، توضیحات کوتاه و کامل
- مدیریت دستورات ربات

### محافظت در برابر اسپم
- محدودیت ۵ پیام در ۵ ثانیه برای هر کاربر
- تشخیص کاربران مسدود شده با نمایش دلیل

## فناوری‌ها

| لایه | فناوری |
|------|--------|
| Runtime | Cloudflare Workers |
| Backend | Hono |
| Frontend | React + React Router DOM |
| UI | Ant Design + Ant Design Icons |
| نمودارها | @ant-design/charts |
| ربات تلگرام | GrammY |
| پایگاه داده | Cloudflare D1 (SQLite) |
| هوش مصنوعی | Cloudflare Workers AI |
| Bundler | Vite + @cloudflare/vite-plugin |
| زبان | TypeScript |
| Linter | OxLint |

## نصب و راه‌اندازی

### پیش‌نیازها
- Node.js 18+
- پنل Cloudflare
- حساب Cloudflare با D1 و Workers AI فعال

### نصب وابستگی‌ها

```bash
pnpm install
```

### ساخت پایگاه داده

```bash
npx wrangler d1 create telegram-bot-dashboard
```

سپس `database_id` دریافتی را در فایل `wrangler.jsonc` جایگزین کنید.

### اجرای مایگریشن‌ها

```bash
npx wrangler d1 migrations apply telegram-bot-dashboard --remote
```

### توسعه محلی

```bash
pnpm dev
```

### استقرار

```bash
pnpm deploy
```

## متغیرهای محیطی و bindings

| Binding | نوع | توضیح |
|---------|-----|-------|
| `DB` | D1 Database | پایگاه داده SQLite |
| `AI` | Workers AI | سرویس هوش مصنوعی Cloudflare |

توکن ربات تلگرام و تنظیمات هوش مصنوعی در جدول `settings` پایگاه داده ذخیره می‌شوند (نه در متغیرهای محیطی) تا در زمان اجرا قابل تغییر باشند.
